## Sobre o Projeto
Este projeto foi desenvolvido para atender a necessidade de gestão centralizada de uma loja de roupas masculina. O sistema substitui planilhas desestruturadas por uma interface intuitiva, focada na organização de dados e auxílio na tomada de decisão.

## Regra de Negócio
O sistema gerencia o fluxo operacional de uma loja, desde a entrada de produtos no estoque até a conclusão da venda ao cliente final. Os processos principais incluem o monitoramento de estoque baixo e a análise de desempenho por meio de relatórios visuais.

## Módulos do Sistema
*   **Dashboard:** Centraliza as métricas de faturamento e volume de pedidos.
*   **Estoque:** Gerenciamento quantitativo com status visual de disponibilidade.
*   **Vendas:** Registro e acompanhamento de transações em tempo real.
*   **Clientes:** Cadastro completo com perfil e preferências de compra.
*   **Relatórios:** Gráficos de vendas anuais e produtos mais rentáveis.

## Diferenciais Técnicos

### Organização do CSS
Para garantir a escalabilidade, o código foi dividido por responsabilidades conforme a estrutura de arquivos visível no projeto:
*   `base.css`: Reset e variáveis globais.
*   `layout.css`: Estrutura de sidebar e navegação.
*   `components.css`: Estilização de botões, cards e tabelas.
*   `responsive.css`: Ajustes específicos para dispositivos móveis.

### Metodologia BEM
Utilizamos a convenção **Block-Element-Modifier** para manter o CSS limpo e evitar conflitos de escopo. 
*   Exemplos aplicados: `.caixa-login`, `.sessao-login`, `.botao-esqueci`.

### Design e UX/UI
O design foi inicialmente prototipado no **Figma** (conforme imagens image_9bfdc1.png e image_911cbe.png), focando em uma estética moderna. Priorizamos a experiência do usuário (UX) com navegação rápida e interface (UI) com cores sóbrias e tipografia legível.

## Tecnologias Utilizadas
*   HTML5 Semântico
*   CSS3 (Variáveis `:root`, Flexbox e Grid)
*   JavaScript
*   **Sem frameworks externos**: Desenvolvimento 100% nativo para maior controle e performance.

## Desafios e Evolução
O maior desafio foi a criação de um layout totalmente responsivo e funcional utilizando apenas CSS puro. Durante a evolução do projeto, refinamos os componentes para garantir que tabelas e gráficos se adaptassem perfeitamente a qualquer tamanho de tela.
